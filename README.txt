САЙТ «НА ДИНАМО» — ВЕРСИЯ 2

КАК ОТКРЫТЬ В VISUAL STUDIO CODE
1. Распакуйте архив.
2. Откройте папку na-dinamo-site-v2 в Visual Studio Code.
3. Откройте index.html.
4. Для полноценного просмотра установите расширение Live Server и нажмите «Open with Live Server».

ЧТО ИЗМЕНЕНО
— установлен официальный логотип из предоставленного архива;
— в светлой шапке используется чёрный логотип, в тёмном подвале — белый;
— все внутренние страницы приведены к единой визуальной системе;
— добавлены одинаковые первые экраны, заголовки, карточки, отступы и кнопки;
— страница «О центре» полностью расширена: описание центра, инфраструктура, форматы занятий и призыв к записи;
— переработаны страницы «Направления», «Тренеры», «Контакты», «Цены» и «Бонусы»;
— удалены встроенные стили из HTML: основное оформление находится в style.css;
— сохранена адаптация для телефонов и планшетов;
— цены на занятия длительностью 1,5 часа отсутствуют;
— форма продолжает отправлять заявки в Bitrix24.

СТРУКТУРА
index.html — главная страница
about.html — о центре
directions.html — направления
pricing.html — цены
coaches.html — тренеры
contacts.html — контакты и форма
loyalty.html — бонусная программа
style.css — всё оформление
script.js — меню, вкладки цен, форма и анимации
images/ — фотографии и официальные версии логотипа

ВАЖНО ПЕРЕД ПУБЛИКАЦИЕЙ
В script.js находится адрес вебхука Bitrix24. На публичном сайте безопаснее перенести отправку формы на серверный обработчик, чтобы вебхук нельзя было увидеть в коде страницы.
